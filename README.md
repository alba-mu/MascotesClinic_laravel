# Autenticación

Esta aplicación permite a los usuarios registrarse, iniciar sesión y cerrar sesión de manera eficiente utilizando las funcionalidades de autenticación de Laravel.

## Rutas Típicas de Laravel
- `GET /register` - Página de registro
- `POST /register` - Envío del formulario de registro
- `GET /login` - Página de inicio de sesión
- `POST /login` - Envío del formulario de inicio de sesión
- `POST /logout` - Cierre de sesión

## Pasos Básicos de Uso
1. **Registro**: El usuario puede acceder a la página de registro para crear una cuenta.
2. **Inicio de Sesión**: Una vez registrado, el usuario puede iniciar sesión en la aplicación utilizando sus credenciales.
3. **Cerrar Sesión**: El usuario puede cerrar sesión haciendo clic en el enlace correspondiente.
